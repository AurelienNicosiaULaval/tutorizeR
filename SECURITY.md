# Security Policy

## Reporting a Vulnerability

If you discover a security issue in tutorizeR, please report it privately by opening an
issue on GitHub with the label `security` and avoiding disclosure of exploit details.

For sensitive disclosures, contact the maintainer directly at:

- `aurelien.nicosia@mat.ulaval.ca`

## Scope

This policy applies to the tutorizeR package and its source code, including scripts,
examples, and CI workflows in this repository.

We review and respond according to severity, then publish public fixes when possible.
